# Logistic Regression

- 知乎：[【机器学习】逻辑回归（非常详细）](https://zhuanlan.zhihu.com/p/74874291)

Logistic 分布是一种连续型的概率分布
- $\mu$: 位置参数
- $\gamma$: 形状参数
>概率分布函数
$$F(x) = P(X \leq x) = \frac{1}{1+e^{-(x-\mu)/\gamma}}$$

>概率密度函数

$$f(x) = F'(X \leq x)= \frac{e^{-(x-\mu)/ \gamma}}{\gamma(1+e^{-(x-\mu)/ \gamma})^2}$$




$$P(Y=1 | x) = p(x)$$
$$P(Y=0 | x) = 1 - p(x)$$
